An end-to-end sales intelligence solution that transforms raw transactional data into actionable insights, enabling forecasting and scenario-based decision-making.

Complete Flow:
 1.  Raw data (CSV)
 2. Python pipeline
      - cleaning
      - feature engineering
      - transformation
 3. Processed dataset
 4. Power BI
      - modeling (DAX)
      - dashboard
      - forecasting
      - what-if analysis
 5. Business insights
